How Live Preview Works

🎨

Change colour

Type a hex code or pick from the colour swatch → header colour updates instantly in the iframe

📤

Upload logo

Drag or click to upload → logo appears in the nav bar of the preview instantly

🔀

Toggle feature

Enable/disable returns, claims, carrier info → preview shows/hides those sections

Data Flow

admin.html (parent page)
  ├── Form inputs capture user changes
  ├── onChange → builds config object
  ├── iframe.postMessage(config) → sends to preview
  └── Config saved to localStorage for persistence

tracking-demo.html (iframe)
  ├── window.addEventListener('message')
  ├── Receives config object
  ├── Applies: colours, logo, fonts, toggles
  └── Updates DOM in real-time
